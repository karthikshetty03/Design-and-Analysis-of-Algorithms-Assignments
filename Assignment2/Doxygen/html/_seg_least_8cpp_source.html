1 /*
2  * Code to implement the line fitting dynamic programming algorithm
3  *
4  *
5  * The cost of a particular set of segments is:
6  * sum(i from 1 to N)
7  * Error_i + cost * amt,
8  *
9  * where,
10  *
11  * Error_i is the error of the ith point of the segment,
12  * cost is the segment cost,
13  * and amt is the number of segments used
14  *
15  * The alogrithm accepts set of points and cost of introducing a segment as input
16  * It outputs the minimal cost and the corresponding set of segments
17  *
18  * Two modes of inputs:
19  * 1. Manually give the number of points say n, and enter n points in n lines
20  *
21  * 2. First run script.cpp file which will take an input say n and generate n*5 points
22  * These points are then written to points.txt file and read directly fron there
23  *
24  * Time Complexity : O(N2)
25  * Space Complexity: O(N2)
26  *
27  */
