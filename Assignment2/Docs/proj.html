Implementation of Segmented Least Squares Algorithm

This algorithm solves a geometrical problem of calculating segmented least square. This is done by minimizing the sum of squares of differences between the segment and the point.




Results produced by our code:

We can see that on changing the cost the space and time complexity is almost the same. Also the number of line segments decreses on increasing the cost in order to find the optimal solution to the problem.
