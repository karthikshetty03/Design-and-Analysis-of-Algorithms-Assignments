Optimal Divide-and-Conquer to Compute Measure and Contour for a Set of Iso-Rectangles

This algorithm solves two geometrical problems of calculating the mesure of the union of a set of iso-rectangles and finding its contour by the measure problem and the contour problem using a single time-optimal divide-and-conquer algorithm. The efficient divide-and-conquer algorithm is obtained by using a technique called "separational representation", which extends the applicability of divide-and-conquer to orthogonal planar objects.

The Measure Problem

Problem statement: In this problem we are given a set of rectangles and we need to find the area of the union of these set of rectangles.

General Discussion on the algorithm: We use divide and conquer method to get the measure of the set of rectangles. We divide Y-axis into multiple stripes and in these multiple stripes we get the x_union, combining both we get the desired result.

The overall runtime of the algorithm is O(N logN)

The Contour Problem

Problem statement: In this problem we are given a set of iso-rectangles and we need to find the contour (boundary) of these rectangles.

General Discussion on the algorithm:

  • We use divide and conquer method to determine the contour pieces. By dividing the Y-axis into multiple stripes and in these stripes we get the set of intervals which will make the contour.
  • For a set of iso-oriented rectangles in the plane, the contour is the boundary between the free and the covered area of the plane. In general it is a collection of disjoint "contour cycles", each of which is a sequence of alternating horizontal and vertical contour pieces
